# ДоброСвайп (DobroSwipe) MVP

ДоброСвайп — это приложение для микро-волонтерства, связывающее некоммерческие организации (НКО) и волонтеров. Волонтеры могут просматривать ленту небольших заданий в формате Tinder-подобных карточек (свайпов), брать их в работу, загружать отчеты и получать карму за выполненные добрые дела. Фонды создают задачи, отслеживают процесс и верифицируют отчеты.

Проект разработан для внедрения в г. Сириус (Адлер).

## Технологический Стек
* **Бэкенд:** Python 3.11, FastAPI, SQLAlchemy (asyncpg), PostgreSQL 15, Redis
* **Фронтенд:** Flutter (Riverpod, GoRouter, Dio, Appinio Swiper)
* **DevOps:** Docker, Docker Compose

## Структура Проекта
* `backend/` - Исходный код API, модели БД, схемы и эндпоинты.
* `frontend/` - Исходный код мобильного приложения Flutter.
* `docker-compose.yml` - Конфигурация для запуска бэкенда, БД и Redis.

## Как запустить бэкенд локально
Убедитесь, что у вас установлен Docker и Docker Compose.

```bash
# Запуск контейнеров в фоновом режиме
docker-compose up -d --build
```
API будет доступно по адресу `http://localhost:8000`. Интерактивная документация (Swagger) доступна на `http://localhost:8000/docs`.

## Как запустить фронтенд
Убедитесь, что установлен Flutter SDK.

```bash
cd frontend
flutter pub get
flutter run
```
